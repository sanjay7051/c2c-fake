# Coorg2Cup Premium Coffee E-commerce Website

## Overview
Coorg2Cup is an e-commerce platform designed to offer premium coffee sourced directly from Coorg, India. The application aims to provide users with a seamless shopping experience for purchasing high-quality coffee products.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/sanjay7051/c2c-fake.git
   cd c2c-fake
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example` and fill it with your configuration.
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:3000` to view the application.
